2. Personalization：自分のデフォルト設定
======================================================================


Personalizationは「自分という利用者の前提」を入れる場所
----------------------------------------------------------------

.. container:: flex

    .. container:: size-1

        **Personalizationとは**

        Custom Instructionsは、ChatGPTに考慮してほしい情報や回答方針を設定でき、全チャットに適用されます。Memoryは、会話をまたいで有用な情報を覚え、回答をより自分向けにできます。

        (OpenAI Help Center)

        つまり  
        Personalizationは「自分という利用者の前提」を入れる場所。

    .. container:: size-2

        .. figure:: _imgs/personalize.png
            :alt: Personalization設定画面
            :width: 100%


例：地方エンジニア向けの設定
----------------------------------------------------------------

.. container:: flex

    .. container:: size-2

        .. revealjs-fragments::
            * 私は地方企業で働くWebエンジニアです。
            * 回答は実務寄りに。
            * 抽象論より、設計判断・運用リスク・小規模チームでの現実解を重視してください。
            * 長崎・九州の現場で「少人数チーム」「兼務が多い」状況を想定してください。

    .. container:: size-1

        .. revealjs-fragments::
            **効果**

            * 毎回説明しなくて済む
            * 自分に合った回答が得られる
            * 一貫性のある回答


3. GPTs：特定用途の専用アシスタント
======================================================================


GPTsは「自分用」ではなく「用途用」
----------------------------------------------------------------

.. container:: flex

    .. container:: size-1

        **GPTsとは**

        特定目的向けに作るカスタム版ChatGPTです。名前、振る舞い、知識ファイル、使えるツールやアクションを設定できます。なお、公式ヘルプではGPTsは保存済みメモリ、カスタム指示、過去会話を使わず、各会話は新しく始まると説明されています。

        (OpenAI Help Center)

        つまり  
        GPTsは「自分用」ではなく「用途用」。

        .. figure:: _imgs/gpts_merged.png

4. Projects：継続作業の作業部屋
======================================================================


Projectsは「案件・テーマごとの作業部屋」
----------------------------------------------------------------

.. container:: flex

    .. container:: size-1

        **Projectsとは**

        関連するチャット、ファイル、カスタム指示をまとめる作業スペースです。長期の調査、執筆、計画、複数チャットにまたがる作業に向いています。公式ヘルプでは、ファイル・指示・チャットを一か所に集めて、ChatGPTが文脈を保ちやすくする機能として説明されています。

        (OpenAI Help Center)

        つまり  
        Projectsは「案件・テーマごとの作業部屋」。
    .. container:: size-1


        .. figure:: _imgs/projects.png
            :alt: Projects画面
            :width: 100%

5. 3つの使い分け
======================================================================


----------------------------------------------------------------

.. container:: flex

    .. container:: size-2

        .. list-table:: 
            :width: 100%
            :header-rows: 1

            * - 機能
              - 何を固定するか
              - 使いどころ
            * - Personalization
              - 自分の前提・好み

              - 特定用途の振る舞い
