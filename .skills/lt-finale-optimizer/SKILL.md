# LTフィナーレ最適化スキル

ライトニングトーク（LT）の最後の部分を印象的に締めくくるための最適化スキルです。

## 概要

LTの最後は「一番記憶に残る場所」であり、「なんとなく終わる」はNGです。このスキルは、LTのテーマや目的に合わせて最適な締めくくり方を提案し、印象的なフィナーレを設計します。

## 主な機能

### 1. 締めくくりパターンの分析
以下の6つの王道パターンからLTの特性に合ったものを提案：
- まとめ＋感謝（王道）
- メッセージの再提示（結論リピート）
- 行動させる（CTA）
- ストーリーで締める（余韻系）
- ビフォーアフター（変化で締める）
- オープニングに戻る（構造美）

### 2. テーマ別最適テンプレート
LTのテーマに応じて、最適な締めくくりテンプレートを提供：
- パッション・情熱系
- 技術・実用系
- ストーリー・エモ系
- 行動喚起系

### 3. NGパターンの検出と改善
以下のようなよくあるNG締めくくりを検出し改善提案：
- 話の流れのまま終わる（唐突）
- 「ご清聴ありがとうございました」だけ
- 自分語りで終わる（聞き手視点なし）

### 4. 個別最適化
LTの内容、目的、聞き手に応じて、具体的な締めくくり文章を生成・最適化します。

## 使用方法

1. LTの概要（テーマ、目的、想定聞き手）を提供
2. 現在の締めくくり案があれば共有
3. 最適なフィナーレパターンと具体的な文章を提案

## 特徴

- **実用性**: すぐに使える具体的なテンプレートを提供
- **状況対応**: LTの目的や聞き手に応じて最適化
- **心理学的アプローチ**: 記憶に残る締めくくりのメカニズムを考慮
- **多様性**: 6つの基本パターンから最適なものを選択

## 対象となるLT

- 技術系ライトニングトーク
- 情熱・思想系プレゼンテーション
- まちづくり・社会貢献系発表
- 個人プロジェクト紹介
- 製品・サービス紹介

## 期待される効果

- 聞き手への印象が3倍向上
- メッセージの記憶定着率が向上
- 行動喚起の成功率が向上
- プレゼン全体の完成度が向上
